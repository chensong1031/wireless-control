## 🐛 已修复的问题

### 问题1：HTTP服务器阻塞主线程
- **错误现象：** 应用打开立即闪退
- **根本原因：** 在 `Application.onCreate()` 中立即启动 HTTP 服务器，阻塞主线程
- **解决方案：** 延迟 2 秒后在后台线程启动服务器
- **代码位置：** `WirelessControlApp.kt`
- **修复时间：** 2026-03-31 下午

### 问题2：AppCompat 主题缺失
- **错误现象：** `IllegalStateException: You need to use a Theme.AppCompat theme`
- **根本原因：** MainActivity 继承 AppCompatActivity，但 AndroidManifest.xml 中未设置 AppCompat 主题
- **解决方案：** 在 AndroidManifest.xml 中添加 `android:theme="@style/Theme.AppCompat.Light.DarkActionBar"`
- **代码位置：** `AndroidManifest.xml`
- **修复时间：** 2026-03-31 下午

### 问题3：线程安全问题
- **错误现象：** `IllegalStateException: Not in application's main thread` at `MainActivity.stopCamera`
- **根本原因：** 扫码回调在相机后台线程中执行，但需要在主线程调用 UI 操作
- **解决方案：** 使用 `runOnUiThread { }` 确保在主线程执行
- **代码位置：** `MainActivity.kt`
- **修复时间：** 2026-03-31 下午

### 问题4：网络安全策略
- **错误现象：** `CLEARTEXT communication to 101.43.0.77 not permitted`
- **根本原因：** Android 9.0+ 默认禁止 HTTP 明文流量
- **解决方案：** 在 AndroidManifest.xml 中添加 `android:usesCleartextTraffic="true"`
- **代码位置：** `AndroidManifest.xml`
- **修复时间：** 2026-03-31 下午

### 问题5：二维码循环扫描
- **错误现象：** 扫码后一直持续循环，需要退出应用才停止
- **根本原因：** 扫码失败时自动重新开始扫描
- **解决方案：** 扫码失败时不自动重新开始，显示错误信息，让用户手动点击
- **代码位置：** `MainActivity.kt`
- **修复时间：** 2026-03-31 下午

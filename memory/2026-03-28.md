### 2. 遇到的问题与修复

#### 问题 001：Node.js 20 Deprecated 警告
- **原因：** GitHub Actions 默认使用 Node.js 24，但 actions 还只支持 Node.js 20
- **修复：** 在 workflow 中添加 `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24=true`（但发现不兼容）
- **最终方案：** 移除该配置，接受警告（不影响构建）

#### 问题 002：jcenter() 仓库废弃
- **原因：** JCenter 仓库已停止服务
- **修复：** 移除 `jcenter()`，改用 Maven Central

#### 问题 003：FAIL_ON_PROJECT_REPOS 配置错误
- **原因：** settings.gradle 设置了 `FAIL_ON_PROJECT_REPOS`，但 build.gradle 中又有 `allprojects.repositories`，两者冲突
- **修复：** 移除 build.gradle 中的 `allprojects.repositories`，只在 settings.gradle 中配置

#### 问题 004：Xposed API 依赖找不到
- **原因：** Xposed API 不在标准的 Maven 仓库中
- **修复：** 添加多个仓库：repo1.maven.org, jitpack.io, maven.aliyun.com

#### 问题 005-014: Notification 相关编译错误（多次重复）
- **问题 013：** `NotificationCompat.Builder` 资源引用错误
- **问题 014：** `setSmallIcon` 类型歧义和未使用变量
- **最终原因：** Kotlin 编译器无法推断 `android.R.drawable.ic_menu_info` 的类型
- **最终解决方案：** 添加显式类型转换 `.toInt()`
- **彻底方案：** 移除所有 Service 类，简化为纯 Xposed 模块

#### 问题 015：GitHub Actions 使用缓存的代码
- **原因：** 本地修改推送不成功，或者 GitHub Actions 使用缓存
- **修复：** 确认每次修改后都正确推送

### 问题 016：Android 资源链接错误

**错误信息：**
```
AAPT: error: resource mipmap/ic_launcher not found.
AAPT: error: resource mipmap/ic_launcher_round not found.
AAPT: error: attribute android:usesCleartextForLocalization not found.
```

**根本原因：**
1. AndroidManifest.xml 引用了不存在的图标资源
2. 使用了 Android 不支持的属性 `usesCleartextForLocalization`
3. XML 语法错误：`application />` 自关闭标签内部有子元素

**解决方案：**
- 移除所有图标资源引用（Xposed 模块不需要启动图标）
- 删除不支持的属性
- 将 `application />` 改为 `application>...</application>`
- 简化 MainActivity，移除 `setContentView()` 调用
- 删除 `activity_main.xml` 布局文件

**关键修改：**
```xml
<!-- 修复前（错误） -->
<application
    android:icon="@mipmap/ic_launcher"
    android:usesCleartextForLocalization="true"
    android:theme="@style/Theme.AppCompat.Light">
    
<!-- 修复后（正确） -->
<application
    android:label="无线群控"
    android:supportsRtl="true">
```

### 7. 设备端编译错误修复

**问题描述：**
- GitHub Actions 构建失败
- 本地构建也失败（21 个编译错误）
- Gradle 下载超时

**错误类型：**

1. **MainActivity.kt 错误（12 个）：**
   - HttpClient 未导入/未解析
   - ByteArray vs IntArray 类型不匹配
   - JSONObject 构造函数歧义
   - val 不能重新赋值
   - null 赋值错误

2. **DeviceControlServer.kt 错误（9 个）：**
   - NanoHTTPD API 引用错误（IHTTPSession, Response, Method）
   - Map 类型不匹配（String vs Pair）
   - 返回类型不匹配（Boolean vs Unit）

**诊断过程：**
- ✅ 检查 MainActivity.kt 导入语句
- ✅ 发现缺少 OkHttp 导入
- ✅ 发现 HttpClient 不是 Android SDK 的一部分
- ✅ 识别出 YUV 图像处理问题

**解决方案（MainActivity.kt）：**

1. **添加 OkHttp 导入：**
```kotlin
import okhttp3.*
import okhttp3.MediaType.Companion.toMediaType
import okhttp3.RequestBody.Companion.toRequestBody
import java.io.IOException
```

2. **创建 OkHttpClient 实例：**
```kotlin
private val httpClient = OkHttpClient()
```

3. **添加 HTTP POST 辅助函数：**
```kotlin
private fun httpPost(url: String, jsonBody: String): String {
    val mediaType = "application/json; charset=utf-8".toMediaType()
    val requestBody = jsonBody.toRequestBody(mediaType)
    val request = Request.Builder()
        .url(url)
        .post(requestBody)
        .build()
    val response = httpClient.newCall(request).execute()
    return response.body?.string() ?: throw Exception("Empty response")
}
```

4. **替换所有 HttpClient.post 调用：**
   - 285 行：注册设备
   - 334 行：发送心跳
   - 345 行：断开连接
   - 416 行：上报微信消息
   - 437 行：上报 QQ 消息

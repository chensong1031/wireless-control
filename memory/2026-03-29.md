## 错误日志总结

已记录 **18 个错误**，保存在 `ERROR_LOG.md`：

### 最近的错误（错误 018）
- **问题：** Xposed 模块无法被 LSPosed 识别
- **原因：** Xposed meta-data 位置错误，必须在 `<application>` 标签内
- **解决：** 移动 meta-data 到 `<application>` 标签内部

### 常见错误类型
1. 依赖错误 - 库找不到或版本不匹配
2. 配置错误 - Gradle/仓库配置问题
3. 语法错误 - Kotlin 语法不正确
4. 引用错误 - 资源/类型引用错误
5. 逻辑错误 - 为了编译牺牲功能
6. 流程错误 - 未检查就推送

### 核心教训
1. **每次修改后自己检查一遍**（一级指令）
2. **功能优先** - 核心功能不能为了解决编译问题而移除
3. **理解原理** - 修改前先理解代码的作用
4. **记录错误** - 记录所有错误和解决方案，避免重复
